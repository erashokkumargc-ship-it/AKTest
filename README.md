# One-Line to One-Page

A simple web app that transforms a short idea into a structured one-page description.

## Features
- Accepts a one-line idea as input
- Generates a polished one-page summary
- Includes sections for overview, benefits, use cases, and conclusion
- Supports keyboard shortcut Ctrl/Cmd + Enter to generate content

## Run locally
Open [index.html](index.html) in your browser, or start a simple local server from this folder.

Example:
```bash
python3 -m http.server 8000
```
